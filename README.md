🎓 Class 10 Science PYQ Interactive Dashboard

Welcome to the Digital Education Report 2026 project! This is a high-performance, single-page application (SPA) designed to help Class 10 students master their board exams through interactive data analysis of Previous Year Questions (PYQs).

🚀 Live Demo

You can view the live website here: https://poonam7083484866-lab.github.io/class-10th-science-pyq-mcqs-/

🌟 Key Features

Trend Analysis: Visualizes the difficulty and frequency of board exam questions over the last 5 years.

Interactive Impact Tracking: Compare different study methods (Digital vs. Traditional) to see which yields higher scores.

Mobile First: Fully responsive design using Tailwind CSS, making it easy to study on smartphones.

Fast Loading: Zero-dependency frontend architecture for lightning-fast performance on slow internet connections.

🛠️ Technical Stack

Structure: Semantic HTML5

Styling: Tailwind CSS (via CDN)

Charts: Chart.js for interactive data visualizations

Deployment: GitHub Pages
